# Wordle Clone

A terminal-based Wordle clone built in Python with color-coded feedback.

## How to Play

1. Clone the repo:

   ```bash
   git clone https://github.com/ioanadaria/wordle-clone.git
   ```

2. Run the game:

   ```bash
   python wordle.py
   ```

3. Guess the hidden 5-letter word in 6 attempts.
   - Green = right letter, right position
   - Yellow = right letter, wrong position
   - Gray = letter not in the word
